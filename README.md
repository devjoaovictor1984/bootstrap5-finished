# Curso de Bootstrap 5 - Projeto Final

Arquivos finais das aulas do curso de Bootstrap 5.

## Como usar

1. Clique em **Code** > **Download ZIP** no GitHub.
2. Extraia o arquivo ZIP no seu computador.
3. Abra os arquivos `.html` no navegador.

Os exemplos usam os arquivos locais em `assets/`, entao nao e necessario instalar dependencias.

## Estrutura

- `01-containers.html` a `28-darkmode-toggle.html`: exemplos finais das aulas.
- `template.html`: arquivo base usado durante o curso.
- `assets/css/`: Bootstrap e estilos adicionais.
- `assets/js/`: Bootstrap Bundle e script do dark mode.
- `assets/images/`: imagens usadas nos exemplos.
